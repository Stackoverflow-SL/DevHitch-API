# DevHitch API

### API Endpoints
BASE_URL = `http://localhost:4000/api/v1`

* Admin
```
POST <BASE_URL>/admin/add
PUT <BASE_URL>/admin/update
DELETE <BASE_URL>/admin/delete/:id
GET <BASE_URL>/admin/get/:id
GET <BASE_URL>/admin/list
```

* Event
```
POST <BASE_URL>/event/add
```
